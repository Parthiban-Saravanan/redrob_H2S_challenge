## ⚡ Key Value Propositions (Why It Wins First Place)

Standard recruitment platforms use basic semantic vector matchings or plain keyword lookups. This application addresses their foundational flaws by deploying an advanced **Three-Tier Hybrid Matrix Framework**:

1. **Contextual Semantic Matching (50% Weight):** Utilizes the `all-MiniLM-L6-v2` transformer model to create 384-dimensional dense vector embeddings of job roles and background profiles, evaluating overall conceptual alignment instead of exact wording.
2. **BM25 Sparse Filtering (30% Weight):** Implements an optimized Okapi BM25 index matching system to guarantee strict enforcement of core tools, program libraries, and languages requested by the engineering team.
3. **Temporal Career Velocity & Stack Density Scorer (20% Weight):** An elite structural assessment algorithm that identifies skill longevity by measuring stack occurrence density while prioritizing recent experiences listed in the initial 30% of candidate profile documentation blocks.
